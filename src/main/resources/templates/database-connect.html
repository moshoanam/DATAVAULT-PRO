Connect Database

Add a new database source to your data catalog

Examples:

  • PostgreSQL: jdbc:postgresql://host:5432/database
  • MS SQL (SQL auth): jdbc:sqlserver://host:1433;databaseName=mydb
  • MS SQL (Windows auth): jdbc:sqlserver://host:1433;databaseName=mydb;integratedSecurity=true;
  • MongoDB: mongodb://host:27017/database

Comma-separated list of collections to extract. Leave empty to extract all collections.
